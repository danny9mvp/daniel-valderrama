Proyectos desarrollados

Prototipo Traductor en Tiempo Real para la Población Sorda en Colombia

Fue mi tesis de grado para aspirar al título de Tecnólogo en Sistematización de Datos en la Universidad Distrital, consistiendo en una aplicación web junto a un sensor Leap Motion para capturar expresiones del lenguaje de señas colombiano y traducirlas a voz mediante un API Text to Speech. Fue nominada a Tesis Laureada. Las tecnologías implementadas fueron: Node.js, jQuery, MongoDB, Leap JavaScript SDK y ResponsiveVoice.js.
